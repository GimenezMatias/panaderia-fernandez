# Imágenes de la Panadería

## Archivos esperados:

1. **exterior.jpg** - Foto del exterior/fachada de la panadería
   - Tamaño recomendado: 1200x800px o similar
   - Formato: JPG o PNG
   - Peso: < 500KB

2. **fachada.jpg** - Foto adicional de la fachada
   - Tamaño recomendado: 1200x800px o similar
   - Formato: JPG o PNG
   - Peso: < 500KB

## Cómo usar en los componentes:

Las imágenes se pueden referenciar desde cualquier componente JSX como:

```jsx
<img src="/images/exterior.jpg" alt="Exterior de la panadería" />
<img src="/images/fachada.jpg" alt="Fachada de la panadería" />
```

## Recomendaciones:

- Comprimir las imágenes antes de agregarlas (usar TinyPNG, ImageOptim, etc.)
- Usar formatos WebP para mejor compresión (si el navegador lo soporta)
- Mantener las imágenes en proporción 4:3 o 16:9 para mejor responsividad
