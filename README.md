# Panaderia-y-Cafe---demo

Demo de página web para Panadería Fernández - Panadería y Café en Córdoba.

Esta aplicación es una SPA (Single Page Application) construida con React, Vite y Tailwind CSS, lista para desplegar en GitHub Pages.

**Características:**
- Página de inicio con hero, productos destacados y información
- Menú completo con categorías
- Información de sucursales
- Página de contacto
- Diseño responsivo

**Desarrollo local:**

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Ejecuta el servidor de desarrollo: `npm run dev`
4. Construye para producción: `npm run build`
5. Previsualiza la build: `npm run preview`

**Despliegue en GitHub Pages:**

1. Sube el contenido de la carpeta `dist/` (generada por `npm run build`) al branch `main` o configura GitHub Actions para automatizar el despliegue.
2. En el repositorio de GitHub, ve a Settings > Pages y configura el source como `Deploy from a branch` con branch `main` y folder `/ (root)`.
3. La aplicación estará disponible en `https://gimenezmatias.github.io/Panaderia-y-Cafe---demo/`.

**Notas:**
- La aplicación funciona completamente sin dependencias externas.
- Las imágenes son placeholders; reemplázalas con imágenes reales según sea necesario.
